Файл ddl содержит скрипт для создания базы данных для задания 2.5
В качестве ответа на первый вопрос создана таблица shops, содержащая столбцы
shop_id и shop_name

В файл dml помещен скрипт main_data_june, содержащий данные для заполнения базы, а также скрипт 
1st_data_mart, собирающий витрину со следующими полями:

shop_name — название магазина,
product_name — название товара,
sales_fact — количество фактических продаж на конец месяца,
sales_plan — количество запланированных продаж на конец месяца,
sales_fact/sales_plan (sales_fact_divide_sales_plan) — отношение количества фактических продаже к запланированному,
income_fact — фактический доход,
income_plan — планируемый доход,
income_fact/income_plan (income_fact_divide_income_plan) — отношение фактического дохода к запланированному.

Параметры расчитаны для каждого наименования товара в каждом магазине

Поскольку план указывается на конец месяца, расчет производится только за июнь (в качестве plan_date указано '2023-06-30')
